# Los_Secretos_Geneticos_de_Springfield_Grupo-11
Repositorio con todos los documentos relativos al proyecto transversal del grupo 11. 

Este proyecto transversal, Los secretos genéticos de springfield, sumerge a los estudiantes en un análisis bioinformático integral utilizando datos ficticios, pero realistas, inspirados en la familia Simpson. A través de tres asignaturas clave del Máster en Bioinformática, exploraremos la expresión génica y las variantes genéticas asociadas a fenotipos completos como la obesidad. Los estudiantes realizarán análisis de expresión diferencial con RNA-seq, anotación de variantes genéticas y aplicarán técnicas bioestadísticas avanzadas, incluyendo PCA y modelos predictivos.

El objetivo es consolidar habilidades prácticas en el manejo de datos ómicos y la interpretación de resultados biológicos, culminando en la presentación de hallazgos en formato de póster científico.
