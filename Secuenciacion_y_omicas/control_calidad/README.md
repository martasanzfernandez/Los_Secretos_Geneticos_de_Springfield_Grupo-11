# Control de calidad

En esta carpeta se incorpora el control de calidad de los datos obtenidos por RNA-seq previos al análisis posterior.
El análisis de calidad se ha realizado mediante  **FastQC** y **MultiQC**, que permiten el análisis individual y en conjunto de los datos, respectivamente.
En esta carpeta se incluye un archivo .html correspondiente a un informe representativo del análisis **FastQC** y el informe obtenido por **MultiQC**.
También se incluyen los scripts utilizados y unas conclusiones generales.
