Se preprocesan los datos a través del programa de Trimmomatic que lo que hará será eliminar 
adaptores que se han utilizado para poder secuenciar la muestra, recorte de bases de baja 
calidad, filtra lecturas cortas o de mala calidad y mantiene la coherencia entre pares, 
generando el archivo paired-end (utilizará para salmon) y separa las lecturas que han quedado
impares en unpaired-end.
